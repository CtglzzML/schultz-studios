# Schultz Studios

**Schultz Studios** is a digital studio focused on building thoughtful products, websites, and digital experiences with strong identity and careful execution.

Created as a shared creative and technical space, the studio brings together design sensitivity, product thinking, and practical development.

---

## About

Schultz Studios is the umbrella brand behind the digital work, experiments, and products created by Carlos Torres and Sara.

The studio is built around a simple idea:

> create digital products with personality, clarity, and care.

Rather than shipping generic work, Schultz Studios is intended to be a place for projects that feel intentional, visually consistent, and well crafted from both the design and technical side.

---

This repository contains the code and structure behind the **Schultz Studios website**, which acts as the digital home of the studio.

## Focus
